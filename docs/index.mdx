Pikku - meaning tiny ğŸ” in Finnish - is a minimalistic batteries included framework built around simple functions. It works with any event driven design (currently **http**, **websockets**, **mcp**, **queues** and **scheduled tasks**) and can be run on any Javascript runtime, currently Cloudflare Workers, Fastify, Bun, AWS Lambda, Express, uWS, WS and others.

## Quick Start

```bash npm2yarn
npm create pikku@latest
```
