# Getting Started with Pikku

Pikku is a TypeScript framework where you write plain functions and wire them to any protocol — HTTP, WebSockets, queues, cron, and more — without changing your business logic.

This guide gets you to a working API in under five minutes.

## 1. Create Your Project

import PackageManagerTabs from '@site/src/components/PackageManagerTabs';

<PackageManagerTabs
  npm="npm create pikku@latest"

Follow the prompts. Choose **Express** if you're unsure — it's the simplest starting point. The CLI will scaffold the project, install dependencies, and generate type-safe clients automatically.

```bash
cd my-app
```

## 2. Start the Server

```bash npm2yarn
npm run start
```

Your API is running at `http://localhost:4002`. Test it:

```bash
curl http://localhost:4002/hello-world
# {"message":"Hello World!"}
```

That response came from a plain TypeScript function in `src/functions.ts`:

```typescript
export const helloWorld = pikkuFunc(
  async ({ logger }) => {
    logger.info('Hello world called')
    return { message: 'Hello World!' }
  }
)
```

Wired to HTTP in `src/wiring.ts`:

```typescript
wireHTTP({
  method: 'get',
  route: '/hello-world',
  func: helloWorld
})
```

The function knows nothing about HTTP. That's the point.

## 3. Add Your Own Function

Open `src/functions.ts` and add:

```typescript
interface GetUserInput {
  userId: string
}

interface GetUserOutput {
  id: string
  name: string
  email: string
}

export const getUser = pikkuFunc<GetUserInput, GetUserOutput>(
  async ({ logger }, { userId }) => {
    logger.info(`Getting user ${userId}`)
    return { id: userId, name: 'John Doe', email: 'john@example.com' }
  }
)
```

Wire it in `src/wiring.ts`:

```typescript
import { getUser } from './functions'

wireHTTP({
  method: 'get',
  route: '/users/:userId',
  func: getUser
})
```

Regenerate types and restart:

```bash npm2yarn
npx pikku && npm run start
```

```bash
curl http://localhost:4002/users/123
# {"id":"123","name":"John Doe","email":"john@example.com"}
```
